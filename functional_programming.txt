Section 4: Functional Programming & Comprehensions

Q16. Explain map, filter, and reduce with examples.
Answer:
map(func, iterable) applies function to every item, returns iterator.
filter(func, iterable) returns items where func returns True.
reduce(func, iterable) from functools reduces iterable to single value.
Example - 
from functools import reduce
nums = [1,2,3,4]
squared = list(map(lambda x: x**2, nums))          # [1,4,9,16]
evens = list(filter(lambda x: x%2==0, nums))       # [2,4]
sum_all = reduce(lambda x,y: x+y, nums)            # 10

Q17. What are list comprehensions? Give an example with nested loops.
Answer:
List comprehensions provide a concise way to create lists. They are often faster than for loops because they use optimized C-level iteration.
Nested loop example: Flatten a matrix:
matrix = [[1,2],[3,4],[5,6]]
flattened = [num for row in matrix for num in row]  # [1,2,3,4,5,6]
With condition: [x**2 for x in range(10) if x%2==0]

Q18. What is a generator and how is it different from a list?
Answer:
A generator yields items one at a time using yield. It doesn't store all values in memory, making it memory-efficient for large sequences.
Generators can only be iterated once.
Example:
def fib_generator(n):
    a, b = 0, 1
    for _ in range(n):
        yield a
        a, b = b, a + b
# vs list: [fib(i) for i in range(n)] would store all values
Use case: Reading huge files, infinite sequences, data streams.

Q19. Explain yield vs return.
Answer:
return ends function execution and returns a single value (or None).
yield pauses function, saves state, and returns a generator. When resumed, it continues from where it paused. 
A function with yield returns a generator object, not a single value.
Example:
def count_down(n):
    while n > 0:
        yield n
        n -= 1
# for i in count_down(5): print(i) -> prints 5,4,3,2,1

Q20. What is itertools? Name three useful functions.
Answer:
itertools is a module for efficient looping and combinatorial operations.
chain(*iterables) – chains multiple iterables.
cycle(iterable) – infinite repetition.
combinations(iterable, r) – all r-length combinations without order.
product(*iterables) – Cartesian product.
Example:
import itertools
list(itertools.combinations([1,2,3], 2))  # [(1,2),(1,3),(2,3)]
